# ระบบ e-Form จัดซื้อจัดจ้าง กฟภ.
**Provincial Electricity Authority — e-Procurement Form System**

---

## 📋 ภาพรวมระบบ

ระบบ e-Form จัดซื้อจัดจ้างของการไฟฟ้าส่วนภูมิภาค (กฟภ.) เป็นเว็บแอปพลิเคชันแบบ Single-File HTML ที่ไม่ต้องติดตั้งเซิร์ฟเวอร์ ทำงานได้ทันทีในเบราว์เซอร์ รองรับการสร้างเอกสารจัดซื้อจัดจ้างทั้ง 3 ประเภทตามระเบียบพัสดุภาครัฐ พ.ศ. 2560 พร้อมแสดงตัวอย่างเอกสารและส่งออกเป็นไฟล์ Word / PDF

---

## ⚙️ ความต้องการของระบบ

| รายการ | รายละเอียด |
|---|---|
| เบราว์เซอร์ | Chrome, Edge, Firefox (เวอร์ชันล่าสุด) |
| การติดตั้ง | ไม่จำเป็น — เปิดไฟล์ HTML ได้ทันที |
| อินเทอร์เน็ต | ต้องการ (ดาวน์โหลดฟอนต์ TH SarabunPSK) |
| ฟอนต์เอกสาร | TH SarabunIT๙ / TH SarabunPSK / TH Sarabun New |
| Google Apps Script | ไม่บังคับ (เชื่อมต่อได้เพื่อบันทึกลง Google Sheets) |

---

## 🗂️ โครงสร้างไฟล์

```
pea_eform.html        ← ระบบหลัก (Single-file HTML)
Code.gs               ← Google Apps Script (สำหรับ Google Sheets/Docs)
README_eForm_PEA.md   ← คู่มือนี้
```

## 🚀 วิธีเริ่มต้นใช้งาน

1. เปิดไฟล์ `pea_eform.html` ด้วยเบราว์เซอร์
2. ระบบจะแสดงหน้า **Dashboard** ทันที ไม่ต้องล็อกอิน
3. กด **+ใหม่** หรือเลือกประเภทเอกสารจากเมนูซ้าย

## 📄 ประเภทเอกสาร

### 1. รายงานขอซื้อ / ขอจ้าง (Form 1)
เอกสารแรกในกระบวนการจัดซื้อจัดจ้าง ใช้รายงานเหตุผลความจำเป็นและขอความเห็นชอบ

**ฟิลด์สำคัญ:**
| ฟิลด์ | คำอธิบาย |
|---|---|
| จาก / ถึง | หน่วยงานผู้ส่ง / ผู้รับ |
| เลขที่ / วันที่ | เลขที่เอกสาร / วันที่ (ระบุเป็นเลขไทยอัตโนมัติ) |
| เรื่อง / เรียน | ชื่อเรื่อง / ผู้รับหนังสือ |
| รายละเอียดพัสดุ | ตารางกรอกรายการ ชื่อ จำนวน หน่วย ราคาต่อหน่วย |
| เหตุผลความจำเป็น | บรรยายเหตุผลที่ต้องจัดซื้อ/จ้าง |
| ราคากลาง | แหล่งที่มาของราคากลาง (ไม่แสดงจำนวนเงิน) |
| วงเงินงบประมาณ | รหัสบัญชี, ศูนย์ต้นทุน, ประเภทงบประมาณ |
| กำหนดส่งมอบ | จำนวนวัน (กรอกเอง) |
| คณะกรรมการตรวจรับ | ชื่อ-ตำแหน่ง กรรมการ 3 คน |
| อำนาจอนุมัติ | ตำแหน่ง, เลขคำสั่ง, วันที่ |
| ผู้ลงนาม / ผู้อนุมัติ | ชื่อ-ตำแหน่ง (แสดงขวามือ เรียงลงมา) |

---

### 2. ขออนุมัติจัดซื้อ / จัดจ้าง (Form 2)
เอกสารขออนุมัติหลังได้รับความเห็นชอบเบื้องต้น พร้อมรายงานผลการเจรจาราคา

**ฟิลด์สำคัญ:**
| ฟิลด์ | คำอธิบาย |
|---|---|
| อ้างอิง | หน่วยงานที่ให้ความเห็นชอบ, ผู้อนุมัติ, วันที่ |
| ผู้เสนอราคา (จำนวนราย) | จำนวนผู้ที่ยื่นเสนอราคา |
| สถานที่ (เลขที่ใบเสนอราคา) | ที่ตั้งผู้ขาย / เลขที่ใบเสนอ |
| จัดซื้อ/จ้างจาก | ชื่อผู้ขาย / ผู้รับจ้าง |
| รายละเอียดพัสดุ | ตาราง 6 คอลัมน์พร้อมสรุป VAT |
| โหมดภาษี | ภาษีนอก / ภาษีใน / ไม่มีภาษี |
| ผู้ลงนาม / ผู้อนุมัติ | แสดงขวามือ, มีช่องว่างลงนาม 70px |

**โหมด VAT:**
- **ภาษีนอก (ex)** — ราคาก่อนภาษี + VAT 7% แยกต่างหาก
- **ภาษีใน (in)** — ราคารวม VAT แล้ว คำนวณถอยหลัง
- **ไม่มีภาษี (none)** — ไม่คิด VAT

---

### 3. คณะกรรมการตรวจรับงาน (Form 3)
เอกสารรับรองการตรวจรับพัสดุ / งานจ้างหลังดำเนินการเสร็จสิ้น

**ฟิลด์สำคัญ:**
| ฟิลด์ | คำอธิบาย |
|---|---|
| วันที่ตรวจรับ | วันที่คณะกรรมการตรวจรับ |
| รายการพัสดุ | คำอธิบายพัสดุที่รับมอบ |
| ผู้ขาย / ผู้รับจ้าง | ชื่อคู่สัญญา |
| จำนวนเงิน | ยอดรวมที่ต้องจ่าย |
| ผู้รับเงิน | ชื่อผู้รับเงิน |
| ประธาน + กรรมการ | ชื่อ-ตำแหน่ง ประธาน 1 + กรรมการ 2 |

## 🔧 Google Apps Script (Code.gs)

### วิธีติดตั้ง

1. เปิด **Google Sheets** ใหม่ → จด Spreadsheet ID จาก URL
2. ไปที่ **Extensions → Apps Script**
3. ลบโค้ดเดิม → วางโค้ดจากไฟล์ `Code.gs`
4. แก้ไข `CONFIG` ในบรรทัดแรก:

```javascript
var CONFIG = {
  SHEET_ID   : 'ใส่ Spreadsheet ID',   // จาก URL ของ Google Sheets
  SHEET_NAME : 'งานจัดซื้อ',
  DOCS_FOLDER: '',                       // Folder ID สำหรับเก็บ Google Docs (ไม่บังคับ)
  ORG_NAME   : 'การไฟฟ้าส่วนภูมิภาค',
  ORG_BRANCH : '',                       // ชื่อสาขา เช่น กฟภ.ชุมแพ
};
```

5. รัน `TEST_setup()` ตรวจสอบว่า Sheet ถูกสร้าง
6. **Deploy → New deployment**
   - Type: **Web app**
   - Execute as: **Me**
   - Who has access: **Anyone**
7. คัดลอก **Web App URL**
8. เปิด **GAS Settings** ในแอป → วาง URL → กด บันทึก

### ฟังก์ชัน GAS ที่มี

| ฟังก์ชัน | หน้าที่ |
|---|---|
| `saveJob(data)` | บันทึก / อัปเดตงานลง Google Sheets |
| `getJobs()` | โหลดรายการงานทั้งหมด |
| `getJobById(id)` | โหลดงานรายการเดียว |
| `deleteJobById(id)` | ลบงาน |
| `createDocument(data)` | สร้าง Google Docs 3 แบบอัตโนมัติ |
